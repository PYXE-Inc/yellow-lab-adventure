# Yellow Lab Spritesheet

## Expected Format

**Filename:** `lab-spritesheet.png`

**Dimensions:** 336x48 pixels (7 frames Ã— 48px width, 48px height)

**Frame Layout (horizontal strip):**
- Frame 0 (0-48px): **Idle** - Lab sitting or standing still
- Frame 1 (48-96px): **Run 1** - First running pose
- Frame 2 (96-144px): **Run 2** - Second running pose
- Frame 3 (144-192px): **Run 3** - Third running pose
- Frame 4 (192-240px): **Run 4** - Fourth running pose
- Frame 5 (240-288px): **Jump** - Lab in air, legs tucked
- Frame 6 (288-336px): **Fall** - Lab falling, legs extended

## Visual Style

- Yellow Labrador dog silhouette/sprite
- Simple, clear poses
- Each frame is 48x48 pixels
- Transparent background (PNG with alpha channel)

## Creating the Spritesheet

You can create this using any image editor:
1. Create a 336x48 pixel canvas
2. Draw 7 frames of the yellow lab in different poses
3. Save as PNG with transparency

Or use the provided generator by opening `spritesheet-generator.html` in a browser.
